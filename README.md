# 🌡️ Temperature Prediction using Machine Learning

## 📌 Overview
This project predicts temperature using environmental and geographical features such as humidity, pressure, latitude, and longitude. It demonstrates a complete machine learning workflow from data preprocessing to model evaluation.
As this was an internship project, I won't be able to provide the CSV file here. I'll share the images of the plots so you can cross-check or review. They can.
---

## 📂 Dataset
The dataset includes:
- Humidity  
- Pressure  
- Latitude  
- Longitude  
- Temperature (target variable)  

---

## 🔍 Exploratory Data Analysis
- Correlation heatmap to understand feature relationships  
- Scatter plot (Humidity vs Temperature) to observe trends  

---

## ⚙️ Model
- Algorithm: Linear Regression  
- Train-test split: 80/20  
- Features used:
  - Humidity
  - Pressure
  - Latitude
  - Longitude  

---

## 📊 Evaluation Metrics
- Mean Squared Error (MSE)  
- R² Score  

---

## 📈 Results
- Model captures general temperature trends  
- Some deviations due to real-world data variability  

---
